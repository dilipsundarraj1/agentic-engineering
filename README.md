# Agentic Engineering

## Introduction

This repository introduces **Agentic Engineering**: a structured way of building software with AI assistants where you stay in control of design, quality, and outcomes. It contrasts unstructured "vibecoding" with a step-by-step workflow focused on clear prompts, small changes, code review, and strong fundamentals.
